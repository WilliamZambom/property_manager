# ROADMAP

## Prazo

Entrega do MVP até 25/03/2025

---

## Fase 1 – Planejamento

- Definir regras de negócio
- Modelar banco de dados
- Definir arquitetura
- Criar estrutura de pastas backend

## Fase 2 – Backend Base

- Configurar Express
- Conectar MongoDB Atlas
- Criar modelos User e Property
- Criar estrutura controllers, services, routes

## Fase 3 – Autenticação

- Implementar bcrypt
- Implementar JWT
- Criar middleware de autenticação
- Proteger rotas privadas

## Fase 4 – CRUD de Imóveis

- Criar rota POST (cadastrar imóvel)
- Criar rota GET público
- Criar rota PUT (editar imóvel)
- Criar rota DELETE (soft delete)
- Implementar validações conforme regras de negócio

## Fase 5 – Upload de Imagens

- Integrar Cloudinary
- Armazenar URLs no banco

## Fase 6 – Frontend

- Criar landing page
- Implementar filtros
- Criar tela de login
- Criar painel admin
- Conectar frontend à API

## Fase 7 – Deploy

- Frontend na Vercel
- Backend no Render
- Banco MongoDB Atlas
- Configuração de variáveis de ambiente

## Objetivo Final

Sistema online antes do fechamento dos primeiros imóveis.
